 Microsoft Asks Web Site to Halt Postings of Code   By By Rebecca Buckman, The Wall Street Journal,  May 12, 2000  In a letter, Microsoft on Wednesday asked the Web site, Slashdot.org, to remove postings from readers containing the specifications for Microsoft's version of "Kerberos," a user-authentication program the company has included in its Windows 2000 operating system.  Microsoft had posted the Kerberos code on its Web site a few weeks ago so software developers-like the ones who read news on Slashdot.org-could review it, a Microsoft spokesman said. But Microsoft also asked people to click on a license agreement before they viewed the code, a move many open-source aficionados thought was close-minded.
